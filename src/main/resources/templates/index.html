Projeto desenvolvido em Java com Spring Boot.

O projeto tem como ídeia a criação de um sistema de registro de pessoas, utilizando os metodos de API Rest com Spring.

Trazendo os métodos HTTP (GET,POST,PUT e DELETE), assim tendo a implementação de uma API completa.

A aplicação também tem a conexão com o MySql aonde vai ser salvo os registro e aonde a API vai ter sua integração com os métodos CRUD

O desenvolvimento da Web Aplicação é criar a API tanto para a utilização do modo em Back End sem inteface gráfica para a registrar as pessoas, tanto utilizando o Front End com a interface gráfica para o usuário.
